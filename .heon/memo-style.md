# Investment Memo Style Guide

Rules the memo must follow. These mirror buyside writing conventions.

## Voice

- **First person plural** ("we believe", "our base case")
- **Direct, not hedging.** "Margins compress 200bps in '27" beats "margins could potentially face some pressure"
- **Confidence calibrated by evidence.** State the view, then state the evidence. Hedge once with a "wrong if" — not in every sentence.
- **One sentence per idea** in narrative sections.

## Banned vocabulary

Never use these words:
- "delve", "delves", "delving"
- "leverage" as a verb ("we leverage X to do Y")
- "robust", "comprehensive", "holistic", "seamless"
- "navigate the landscape", "navigate the challenges"
- "in today's fast-paced", "in an ever-evolving"
- "stands out", "sets itself apart"
- "deep dive", "unpack", "double-click"
- "moreover", "furthermore", "additionally"
- "It's worth noting that" (just say the thing)

## Banned constructions
- **Tricolons** — three-item lists for rhythmic effect
- **Em dash abuse** — at most one per paragraph
- **"Not just X but Y"** — overused contrast structure
- **Generic closing summaries** — end on a specific claim or tripwire
- **Hedge stacking** — "could potentially possibly maybe"

## Numbers
- **Quantify everything.** "Margins expanded 380bps" not "margins improved"
- **Percentages with explicit base.** "Revenue grew 28% YoY"
- **Multiples in line.** "Trading at 18x FY26 EPS"
- **Don't invent numbers.** If you don't have it, write "not disclosed"

## Thesis bullet format

Every thesis bullet must be a complete falsifiable claim:

> **[Specific claim with a number]** — [Evidence with a specific number]. *Wrong if [specific observable would happen].*

## Bear case requirements

1. **Steelmanned.** Written as if you actually believed it.
2. **Specific competitor / customer / regulator named.**
3. **Coherent world, not a list.**
4. **If bear reads weaker than bull, rewrite.**

## Tripwires

Every risk row must have an observable tripwire:
- Good: "Q3 ad-tier ARPU prints below $8"
- Good: "Gross margin contracts more than 150bps in any quarter"
- Bad: "Macro deteriorates"
- Bad: "Competition increases"
